Turn Assist Light

Installation

If your Gogoro Smartscooter™ is not equipped with Turn Assist Lights, you may purchase them at Gogoro Service Center and ask the technicians to install them for you.

Activation

When the Turn Assist Lights are installed, Gogoro Service Center will set up server authentication (about 10 minutes needed).

  • After authentication is confirmed, the new function of choosing lighting pattern will show up in the Gogoro App after refresh. There is no need to restart the App or log in again.
  • The Turn Assist Lights will start working after your next battery exchange.
  • If a user exchanges batteries soon after the Turn Assist Lights are installed, it is possible that the server is still processing the authentication and has not yet transferred data to the batteries. In this case, the user will have to exchange batteries again a few minutes later.

Lighting behavior

Smartscooter™ Status : Unlocked

  • Daytime (Sufficient ambient light) : Flash along with the unlock animation and then turn off.
  • Night time (dark environment) : On

Smartscooter™ Status:Motor is activated, on the run, motor is off and turns into unlock mode

  • Daytime (Sufficient ambient light):Off
  • According to user’s Gogoro App setting
    • Mode 1 (default): Normally off. When the turn signal is flashing, the Turn Assist Light on the same side will be on constantly.
    • Mode 2 : Always on.
    • Mode 3 : Normally on, when user hits the brake and stops, Turn Assist Lights will be flashing along with the pulsating frequency of LED position lights.

Smartscooter™ Status:Locked

  • Daytime (Sufficient ambient light):Off
  • Night time (dark environment):
    • Off
    • If user has set the delayed headlight off function, it will be off along with the headlight after set amount of seconds.
